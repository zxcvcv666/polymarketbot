## 1. 重新生成 Safe 钱包

如果你想为新账户重新生成 Safe 钱包，需要执行以下步骤：

### 步骤 1：准备新的 Builder 凭证

1. 登录 [polymarket.com](https://polymarket.com)
2. 进入 **Settings → Builder** 标签页
3. 点击 **Create New API Key** 创建新的 Builder 凭证
4. 记录下 `API Key`、`Secret`、`Passphrase`

### 步骤 2：更新 .env 配置

编辑 `/opt/test/polymaker_bot/.env`：

```bash
# 新的 Builder 凭证
POLY_BUILDER_API_KEY=新的API_KEY
POLY_BUILDER_SECRET=新的SECRET
POLY_BUILDER_PASSPHRASE=新的PASSPHRASE

# 签名类型设为 2 (Safe 钱包)
SIGNATURE_TYPE=2

# 地址先留空或设为新 Safe 地址（部署后会获得）
ADDRESS=0x新Safe地址
FUNDER_ADDRESS=0x新Safe地址
```

## 3. 彻底关闭脚本

### 方法 1：通过命令行

```bash
# 查找并杀掉所有相关进程
pkill -9 -f "python main.py"

# 确认进程已终止
ps aux | grep "python main.py" | grep -v grep
```

### 方法 2：使用 PID 文件

```bash
# 如果有 PID 文件
cat /opt/test/polymaker_bot/polymaker.pid

# 根据PID杀掉进程
kill -9 $(cat /opt/test/polymaker_bot/polymaker.pid)

# 清理 PID 文件
rm /opt/test/polymaker_bot/polymaker.pid

```

### 方法 3：完整清理脚本

```bash
#!/bin/bash
# 停止所有 polymaker 相关进程

echo "正在停止 polymaker 机器人..."

# 方法1: 通过进程名
pkill -9 -f "polymaker" 2>/dev/null

# 方法2: 通过 Python main.py
pkill -9 -f "python main.py" 2>/dev/null

# 清理 PID 文件
rm -f /opt/test/polymaker_bot/polymaker.pid

# 确认
if pgrep -f "python main.py" > /dev/null; then
    echo "警告: 仍有进程在运行"
    ps aux | grep "python main.py" | grep -v grep
else
    echo "✅ 所有进程已终止"
fi
```
